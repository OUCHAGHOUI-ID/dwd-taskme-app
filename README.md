# dwd-taskme-app
Une application web simple de gestion de tâches, construite avec **HTML**, **CSS** et **JavaScript**.
Cette application permet de créer, visualiser et gérer des tâches directement depuis le navigateur.

## 🚀 Fonctionnalités
- Ajouter, modifier et supprimer des tâches
- Interface utilisateur responsive
- Thème clair / sombre (si implémenté dans themeMode.js)
- Notifications pour certaines actions (notification.js)
- Barre latérale dynamique (sidebar.js)

## 💻 Installation et utilisation
1. Cloner le dépôt :
   git clone https://github.com/OUCHAGHOUI-ID/dwd-taskme-app.git
2. Ouvrir le projet :
   Double-clique sur index.html ou ouvre-le dans ton éditeur préféré.
3. Commencer à gérer tes tâches !

## 🛠️ Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla JS)
